# COMP120 Version Control Project

## Description
This project is a simple task manager that allows users to add and view tasks.

## Features
- Add a task
- View tasks

## How to Use

1. Clone the repository:
   git clone https://github.com/vithusanvara/comp120_assignment3.git

2. Open the project folder

3. Run the program:
   python app.py

## Example Output
Task added: Complete assignment
Your Tasks:
- Complete assignment
